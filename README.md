# Drift Extension

Drift is a behavioral interruption extension that gently checks in during social media use.

## Install (Unpacked Extension)

1. Download the repo ZIP and unzip it.
2. Open Chrome and go to `chrome://extensions`.
3. Turn on **Developer mode** (top right).
4. Click **Load unpacked**.
5. Select the unzipped folder that contains:
   - `manifest.json`
   - `content.js`
   - `background.js`
   - `overlay.css`

The extension will be available immediately.

## Enable Site Access

1. Go to `chrome://extensions`.
2. Click **Details** on Drift.
3. Under **Site access**, choose **On specific sites**.
4. Add these:
   - `https://www.youtube.com/*`
   - `https://www.tiktok.com/*`
   - `https://www.instagram.com/*`
   - `https://twitter.com/*`
   - `https://x.com/*`

## Update

If you pull a new version, click **Reload** on the extension in `chrome://extensions`.
